5. Server pair bonding

Disconnect from the server by running command exit
Create an ssh keypair between your linux machine and the remote server
Run command ssh-keygen which will generate a ssh key pair
ssh-keygen will ask you several questions, skip them by pressing ENTER
Run command ssh-copy-id USER@HOSTNAME to copy the public ssh key to the remote server
It will ask you to provid admin's passworrd: tuxedo9999 Try connecting to remote server with SSH, this time it shouldn't ask you for a password
